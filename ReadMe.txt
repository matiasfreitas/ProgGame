Para a aula de PROG no MIEIC FEUP.

Atingimos todos os objetivos propostos, usando classes/objetos mais relacionadas com as peças do tabuleiro (“Tiles”) do que com as palavras (“words”). O jogo deve funcionar corretamente nos sistemas operativos Windows e Linux.

Fizemos um melhoramento, alteramos as cores das letras no tabuleiro de acordo com o seu estado. Assim: 
-Se um jogador já colocou uma das suas peças sobre uma letra, ou seja, se não pode jogar mais naquela letra, esta estará com cor vermelha; 
-Se uma letra pode ser uma possível jogada esta estará com cor verde; 
-Todas as outras letras estarão com cor preta.

Regras do jogo: https://www.hasbro.com/common/instruct/Scrabble_Junior.pdf
